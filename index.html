What is Mono?

Mono is an open source, cross-platform implementation of C# and the CLR that is binary compatible with the .NET Framework. Mono has been in development since 2001 and is very mature.

Here's how it all started as told by Miguel de Icaza.

The Mono Project is sponsored by Xamarin, who now have a worldwide strategic partnership with Microsoft.

What is Tizen?

Tizen is an open source, standards-based software platform supported by leading mobile operators, device manufacturers, and silicon suppliers for multiple device categories such as smartphones, tablets, netbooks, in-vehicle infotainment devices, and smart TVs.

SEE: The Tizen Project is broken and we will be spending some time apart.

What is MonoTizen?

MonoTizen is a port of Mono to Tizen, including an IDE integration for MonoDevelop.

How do I use MonoTizen?

  • Install the MonoDevelop.Tizen add-in.
  • Install Mono-3.6.0 RPMs on a Tizen device
