What is Mono? What is Tizen?

Mono is an open source, cross-platform implementation of C# and the CLR that is binary compatible with Microsoft.NET

Tizen is an open source, standards-based software platform supported by leading mobile operators, device manufacturers, and silicon suppliers for multiple device categories such as smartphones, tablets, netbooks, in-vehicle infotainment devices, and smart TVs.

What Tizen devices are available or coming soon?

  • Samsung Gear 2 smartwatch
  • Samsung Z smartphone
  • Tizen TV
  • Jaguar Landrover IVI

How do I use MonoTizen?

  • Install MonoDevelop or Xamarin Studio.
  • Install the MonoDevelop.Tizen add-in.
  • Have fun!
